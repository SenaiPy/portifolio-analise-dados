# Explicação das tabelas

## Cadastro*
Todas as tabelas cadastro (clientes, localidades, lojas e produtos) são do tipo dimensão, representam características únicas de acordo com cada tabela.

## Vendas*
Todas as tabelas vendas (2020, 2021 e 2022) são do tipo fato, contém um histórico de todas as vendas concluídas no ano referente a cada tabela.

## Devoluções
Tabela do tipo fato, contém um histórico de todas as devoluções feitas durante todo o período (2020, 2021 e 2022).
